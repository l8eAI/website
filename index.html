Start Building Today with `l8e-beam`.

`l8e-beam` is our foundational Python SDK for PII sanitization. It's the perfect entry point to building secure AI applications on the Lighthouse platform. 

$ pip install l8e-beam

For "set-and-forget" protection, just add the decorator to any function.

from l8e_beam import redact_pii, PiiAction

@redact_pii(action=PiiAction.ANONYMIZE)
def create_test_user(profile: dict):
    return profile

user_profile = {
    "name": "John Doe", 
    "email": "john.doe@example.com"
}
anonymized = create_test_user(user_profile)

The decorator automatically processes inputs and outputs.

// Output
{
  "name": "Mary Smith", 
  "email": "robertholmes@example.org"
}

For dynamic processing and full control over the sanitization rules.

from l8e_beam import sanitize_pii, DefaultRecognizer

report = "Contact support@acme.com about the issue with John Smith."

# Redact names, but leave emails untouched
processed = sanitize_pii(
    report,
    disabled_recognizers=[DefaultRecognizer.EMAIL]
)

The API gives you fine-grained control.

// Output
"Contact support@acme.com about the issue with [PERSON]."
